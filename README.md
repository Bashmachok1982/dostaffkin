# Dostaffkin

Легкий Angular-приложение для расчета доставки и оформления заявок на курьерскую доставку с картами Яндекс.

## О проекте

Приложение позволяет выбрать маршрут, тип посылки и скорость доставки, рассчитать стоимость и оформить заявку на доставку через API `https://testologia.ru/delivery/create`.

## Основной функционал

- Страница заказа: ввод адресов "Откуда" и "Куда"
- Автозаполнение адресов через Яндекс suggest
- Построение маршрута и расчет расстояния на карте
- Расчет стоимости по тарифам размера и скорости
- Отправка заявки с данными клиента
- Страница отслеживания и домашняя страница

## Технологии

- Angular 21
- TypeScript
- RxJS
- ngx-toastr
- Yandex Maps API
- NPM

## Запуск

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```

## Структура

- `src/app/pages/order` — страница заказа с маршрутом и формой
- `src/app/services/delivery-api.ts` — API для создания заявки и получения статуса
- `src/index.html` — подключение Яндекс.Карт
- `docs/` — статический вывод сборки
